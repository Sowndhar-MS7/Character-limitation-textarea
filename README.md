# Character-limitation-textarea

# ✍️ Textarea with Real-Time Character Counter

This project demonstrates a **simple textarea input** with a **real-time character counter** built using **HTML, CSS, and JavaScript**. The counter updates as the user types, showing both the number of characters typed and the remaining characters allowed.

---

## 🚀 Features
- ✅ Textarea input field with a **200 character limit**.  
- ✅ **Real-time character counter** (`x/200 characters`).  
- ✅ Displays a **warning message** when the limit is reached.  
- ✅ Prevents the user from typing beyond the maximum allowed characters.  
- ✅ Clean and responsive design with simple CSS.
